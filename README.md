## 🛡️ Sobre Nosotros

ShadowNetForge es una comunidad open-source dedicada a crear herramientas gratuitas para privacidad digital y acceso sin restricciones a internet.

Nuestra misión:
- Evadir censura y bloqueos geográficos
- Proteger datos personales al máximo
- Facilitar navegación anónima
- Simplificar el self-hosting de VPN y proxies

🚫 Sin logs  
🚫 Sin anuncios  
🚫 Sin suscripciones  
✅ Solo código abierto y comunidad
