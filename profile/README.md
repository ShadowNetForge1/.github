## Core Philosophy

We build tools that disappear after use.  
No cloud dependencies.  
No user tracking.  
No corporate oversight.

Focus areas:
- WireGuard & modern tunneling protocols
- Configuration automation & optimization
- Easy self-hosting on low-end hardware
- Bypass resistance against DPI / censorship
